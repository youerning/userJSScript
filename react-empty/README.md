#### 安装教程

1. 安装依赖     
```
# 安装依赖包
yarn install
```
2. 打包
```
#普通打包, 不适用于生产环境
yarn run dist
# 生产环境打包
yarn run disp
```
3. 测试
```
#启动webpack-dev-server测试服务器用于测试
yarn run dev
```
